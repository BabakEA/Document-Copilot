## 9. Appendix – Technical Stack
- **API:** FastAPI (Python), containerized via Docker
- **GUI:** Streamlit (Docker)
- **Database:** ChromaDB vector database (customizable)
- **Embedding Model:** all-mpnet-base-v2 (customizable)
- **Referencing System:** AI-Forest PateLLM (OpenAI, T5, LLaMA2, LLaMA3)
- **Bidirectional Hybrid Multi-head Classifier**

---
## 10. Document Copilot Architecture Overview
1. **User Registration:** Generates a unique access token
2. **Document Upload:** Converts documents (PDFs, images, text, etc.) into vectorized data
3. **Q&A Function:** Uses RAG to retrieve relevant information
4. **Research Copilot:** Generates entire research papers from project descriptions using the vector database
